# 💻 Segunda prova de estrutura de dados - UFPA 

Este repositório reúne a resolução da prova pratica de estrutura de dados desenvolvida com a linguagem Python. O objetivo é consolidar conhecimentos de python e Programação Orientada a Objeto, por meio de desafios e problemas práticos.

## Questão
<p style="text-align: center>
Uma biblioteca comunitária deseja informatizar parte do seu controle de empréstimos e devoluções de livros. Para isso solicita um programa simples que permita o registro da quantidade de livros atualmente disponíveis e dos que foram emprestados. Quando novos livros chegam ao acervo, eles devem ser adicionados ao total disponível. Quando um usuário pega livros emprestados, essa quantidade deve ser removida do estoque disponível e somada ao total de livros emprestados. Da mesma forma, quando livros são devolvidos, eles voltam ao estoque da biblioteca, reduzindo o número de livros emprestados. A biblioteca também deseja poder visualizar a situação atual do acervo a qualquer momento com o nome da instituição, a quantidade de livros disponíveis e total de livros emprestados. O sistema deve funcionar por meio de um menu interativo simples, no qual o usuário possa escolher as opções de adicionar livros ao acervo, emprestar e devolver e exibir situação atual, repetindo essas opções até decidir sair do programa.
</p>   

## 🛠️ Como Usar

1. Clone este repositório:

   ```bash
   git clone https://github.com/Dev-Joao-Medeiros/Prova2-Estruturadedados.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd Prova2-Estrutura-de-dados
   ```

3. Escolha a atividade desejada e execute com:
   ```bash
   python nome_do_arquivo.py
   ```
